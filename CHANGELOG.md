# Pedy Website Changelog

## [Unreleased]

## [2.0.0] - 2024-12-19

### Added
- **Complete project restructure** with organized folder structure
- **Accessories page** (`pages/accessories.html`) with:
  - Product grid showcasing 6 tech accessories
  - Category filtering (All, Mobile, Laptop, Audio, Gaming)
  - Sort functionality (Price, Popularity, Newest)
  - Product cards with hover effects and action buttons
- **News page** (`pages/news.html`) with:
  - Latest tech news articles with metadata
  - Trending topics sidebar
  - Category filtering system
  - Newsletter subscription section
- **Resources page** (`pages/resources.html`) with:
  - Educational content categories
  - Learning resource cards for tutorials and guides
  - Resource type filtering
  - Professional course listings
- **Contact page** (`pages/contact.html`) with:
  - Contact form with validation
  - Business information cards
  - FAQ section with 6 common questions
  - Map placeholder and contact details
- **Admin dashboard** (`admin/index.html`) with:
  - Modern admin interface design
  - Sales and order statistics
  - Recent orders management table
  - Activity feed with real-time updates
  - Responsive sidebar navigation
- **Organized assets structure**:
  - `assets/css/main.css` (moved from styles.css)
  - `assets/js/main.js` (moved from script.js)
  - `assets/images/logo.png` (moved from root)
- **Mobile hamburger menu** toggle functionality
- **Floating emoji elements** on hero section
- **Feature cards with icons** in hero section
- **Enhanced text animations** and effects
- **Cross-page navigation** linking all sections

### Changed
- **Restructured entire project** with proper folder organization
- **Updated all file paths** to use new asset structure
- **Enhanced navigation** to include all new pages and admin access
- **Improved responsive design** across all pages
- **Consistent design language** throughout all pages
- **Professional styling** with modern UI/UX principles
- **Updated README.md** with comprehensive project documentation
- Website focus from academic resources to comprehensive tech hub
- Navigation links to English (removed Swahili from nav)
- Kept Swahili elements only on landing page hero section
- Enhanced hero title with glow effects and gradient text
- Improved mobile logo positioning and responsive design

### Fixed
- **File organization** issues with scattered assets
- **Navigation consistency** across all pages
- **Responsive design** breakpoints for all pages
- **Mobile compatibility** for all new pages
- Mobile navigation layout issues
- Search bar positioning on mobile
- Hero section padding for mobile header
- Logo sizing and positioning across devices

### Technical Improvements
- **Semantic HTML structure** for better accessibility
- **CSS Grid and Flexbox** for modern layouts
- **Consistent color scheme** using CSS custom properties
- **Font Awesome icons** integration throughout
- **Google Fonts** typography implementation
- **Form validation** and interactive elements
- **Modular CSS architecture** for maintainability
